# UnMarket 规则约束文档

## 一、养号规则

### 1.1 账号生命周期

| 阶段 | 状态 | 说明 |
|------|------|------|
| new | 新建 | 刚添加的账号，尚未开始养号 |
| warming | 预热期 | 正在养号中，每天需完成浏览任务 |
| active | 活跃期 | 养号完成，可正常发布内容 |

### 1.2 各平台养号策略

| 平台 | 预热天数 | 每日次数 | 单次时长 | 活跃时段 |
|------|---------|---------|---------|---------|
| Twitter/X | 14天 | 2-4次 | 3-10分钟 | 08:00-23:00 |
| Reddit | 14天 | 2-3次 | 5-15分钟 | 09:00-22:00 |
| LinkedIn | 7天 | 1-2次 | 3-8分钟 | 08:00-20:00 |
| 知乎 | 14天 | 2-4次 | 5-15分钟 | 09:00-23:00 |
| 小红书 | 14天 | 3-5次 | 5-15分钟 | 10:00-23:00 |

### 1.3 养号行为规范

养号期间只执行以下行为：
- 浏览首页/推荐内容
- 随机滚动页面
- 模拟阅读停顿
- 偶尔悬停在帖子上

**禁止行为：**
- 发帖
- 评论
- 点赞/收藏（前3天）
- 关注账号（前3天）


### 3.3 回复内容规范

**自然回复模板示例：**
- "I've been using {产品} for this exact use case. {tagline}. Might be worth checking out!"

**回复原则：**
- 内容要自然，避免硬广告
- 根据帖子主题调整措辞
- 不要每条都提产品链接
- 语气要像真实用户分享经验

## 六、风控红线

### 6.1 绝对禁止

- 在预热期未满时频繁发帖
- 短时间内大量回复
- 内容高度重复/相似
- 使用同一 IP 操作多个账号
- 绕过平台的发帖限制

### 6.2 高风险行为

- 新账号立即发广告
- 回复内容千篇一律
- 发帖时间过于规律
- 只发不互动
- 频繁发送链接

### 6.3 安全建议

1. 每个账号使用独立浏览器 Profile
2. 养号期间保持耐心，不要急于发帖
3. 内容要有价值，不是纯广告
4. 适当参与社区讨论，不只推广
5. 遵守各平台社区规范
